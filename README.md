 Generador de Nombres - Proyecto IA #2

Este proyecto genera nombres automáticos estilo startup o empresa tecnológica utilizando combinaciones aleatorias de palabras.

Forma parte de mi desafío personal de desarrollar 100 proyectos con Inteligencia Artificial y programación en Python.

---

 Tecnologías utilizadas

- Python
- Librería random
- Estructuras de datos (listas y set)

---

 ¿Cómo funciona?

El programa:

1. Tiene listas con palabras iniciales, medias y finales.
2. Combina aleatoriamente esos elementos.
3. Genera nombres únicos.
4. Evita repeticiones usando un `set`.

Ejemplo de salida:

Tech Labs AI  
Neo Systems Pro  
Cyber Logic Hub
